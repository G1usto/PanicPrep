# PanicPrep Context

## Goal
PanicPrep is a mobile-first student panic helper app.

Students upload homework/worksheet screenshots and choose a mode:
- Explain Step-by-Step
- Simplify
- Summarize
- Generate Quiz

## Budget
No paid AI APIs for MVP.
Only use GPT, Claude, and Codex subscriptions manually.

## MVP Rule
Build the UI and workflow first with mock responses.
Later add real AI only after validation.

## Tech
- Web app / PWA first
- React or Next.js
- Tailwind CSS
- localStorage first
- no login
- no database
- no payments

## Current Build Goal
Create mobile-first UI:
- upload image
- image preview
- mode buttons
- mock AI result
- copy prompt for ChatGPT/Claude
- local history

## Do NOT Build Yet
- accounts
- Stripe
- real AI API
- social features
- native mobile app
- PDF support
